# Phase 0 — Project Skeleton + Deployment

## Goal
Get a blank Next.js app deployed to Vercel. Nothing functional yet — just a working foundation.

## Deliverables
- Next.js 14+ app created with App Router
- Tailwind CSS configured
- shadcn/ui installed and working
- Pushed to GitHub
- Deployed to Vercel
- Basic navbar with "Pitchpilot" branding
- Single landing page with placeholder text

## Steps
1. Run `npx create-next-app@latest pitchpilot --typescript --tailwind --app`
2. Install shadcn/ui: `npx shadcn-ui@latest init`
3. Create a basic navbar component
4. Create a minimal landing page (just headline + subheadline, no functionality)
5. Push to GitHub
6. Connect repo to Vercel and deploy

## Done When
- `pitchpilot.vercel.app` (or similar) loads in browser
- No console errors
- Navbar renders correctly
- Tailwind styles are working
